🌲 Timber!! – A Simple 2D Reflex Game in C++
🪓 Description
Timber!! is a beginner-friendly 2D reflex game built with C++ and SFML. Inspired by the classic mobile game, you play as a lumberjack chopping a tree, dodging falling branches, and racing against the clock.

This project is from the Beginning C++ Programming by John Horton (not the mathematician) and focuses on object-oriented design, SFML basics, and real-time input handling.

📸 Gameplay Overview
Press Left Arrow or Right Arrow to chop.

Time decreases steadily; chopping gives you more time.

Avoid branches that come down on your side.

Game over if you run out of time or get hit.

🧰 Features
Real-time keyboard input

Simple UI: Score, timer, and FPS

Branch generation logic

Sprite animation

Sound effects

Cross-platform with SFML

🧠 Concepts Covered
C++ Basics: Loops, conditionals, functions

SFML: Windowing, events, sprites, text, and sound

Object-Oriented Programming // not really but try to implement this game using a OOD approach

Game loop and timing with sf::Clock

Vector math for positioning

🛠️ Requirements
C++ compiler (GCC, Clang, MSVC)

SFML 2.5+

▶️ How to Run
Install SFML

Compile the code:
  g++ -o Timber main.cpp -lsfml-graphics -lsfml-window -lsfml-system -lsfml-audio
run the executable:
  ./Timber
